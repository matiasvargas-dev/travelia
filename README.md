# Travelia

Proyecto de página web estática para la materia PYSW: Programación y Servicio Web.

- Alumno: Matías Emanuel Vargas
- LU: 4810

## Descripción

Travelia es un sitio web de agencia de viajes creado con HTML y CSS. Incluye una página principal y varias vistas secundarias con contenido estático, navegación y diseño responsivo.

El nombre, el logo y la paleta de colores del proyecto están disponibles en `assets/img`.

## Estructura del proyecto

- `index.html` - Página principal.
- `views/` - Vistas secundarias.
  - `destinos.html`
  - `agencias.html`
  - `contactos.html`
  - `precios.html`
  - `blog.html`
- `styles/styles.css` - Estilos globales.
- `assets/` - Recursos estáticos (imágenes, videos).

## Contenido principal

- Menú de navegación con enlaces entre las secciones.
- Página de destinos con tarjetas y comparador de precios.
- Página de agencias con fichas de proveedores.
- Página de contacto con formulario.
- Página de precios con tabla comparativa.
- Página de blog con artículos y comentarios.

## Observaciones

- El proyecto es estático y no incluye backend.
- Está pensado para presentación del trabajo práctico, con diseño y contenido enfocado en una agencia de viajes.
